RULES

進行
各自スマホで同じルームコードに JOIN (最大8人)。ホストが START したら全員に1枚配られる。自分のカードは見えない、他人のは見える。
宣言フェーズ
順番に「全員のカード合計は ○○ 以上」と宣言する。次の人は 前より大きい数 で宣言し直すか、「BLUFF!」 で挑戦する。
BLUFF 宣言
挑戦時、全員のカードを公開して実合計を計算:
• 実合計 ＜ 宣言値 → 宣言した人 が負け (ハッタリ)
• 実合計 ≥ 宣言値 → 挑戦した人 が負け
負けた人が 1杯、次ラウンドの開始者。
特殊カード
• ×2: 全員のカード合計を2倍 (×2 が複数なら掛け合わせ)
• MAX→0: 場で一番大きい数値カードを 0扱い にして合計
• ?: 山から1枚追加で引いて場に加わる (連鎖あり)
計算順: 数値合計 → MAX→0 → ×2
カード構成
35枚: −10/−5/0/1/2/3/4/5/10/15/20 + 特殊3種 (×2, MAX→0, ?)
